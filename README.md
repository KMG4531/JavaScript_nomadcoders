# JavaScript_nomadcoders
노마드코더 자바스크립트 강의

항상 끝에 **;** 기억

변수 상수
```javascript
const a = 5; // 변하지 않는 값
let b = 5; // 변해도 되는 값
```

콘솔
```javascript
console.log(); // 콘솔에 띄워줌
```
log는 print랑 똑같이 생각하면 됨

배열(Array)
```javascript
const dayOfWeek = ["mon", "tue", "wed", "thu", "fri", "sat"];
console.log(dayOfWeek) // 콘솔에 배열인 dayOfWeek가 출력

// 배열 안에 있는 것을 출력하고 싶으면
console.log(dayofWeek[4]);  // 배열에서 fri이 출력
```
